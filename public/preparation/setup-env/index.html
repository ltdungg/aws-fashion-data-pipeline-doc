2.2 Create VPC, Subnet, ...

Create VPC (Virtual Private Cloud)

  1. Access the Amazon Management Console
    • Search for the VPC service
    • Select VPC from the search results

  1. In the VPC interface, select Create VPC

  1. In the Create VPC interface:
    • Choose VPC and more
    • For name tag auto-generation, enter: fashion
    • IPv4 CIDR block: 10.10.0.0/16

  1. Leave the remaining settings as default, and select Create VPC at the bottom of the screen.
